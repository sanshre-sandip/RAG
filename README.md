
# RAG (Retrieval-Augmented Generation) Exploration

This repository contains various experiments and implementations related to Retrieval-Augmented Generation (RAG). It covers the entire pipeline from document loading to chunking, vector storage, and retrieval.

## Repository Structure

- **[Document-Loaders/](./Document-Loaders/):** Scripts and notebooks for loading different document formats (CSV, PDF, TXT, Web).
- **[Chunking/](./Chunking/):** Various strategies for splitting documents into smaller chunks (Length-based, Semantic, Structure-based).
- **[Vectors_storing/](./Vectors_storing/):** Implementations for converting chunks into embeddings and storing them in vector databases.
- **[Retriever/](./Retriever/):** Methods for retrieving relevant information from the vector store or external sources like Wikipedia.

## Getting Started

Each folder contains specific implementations and examples in Jupyter Notebooks. Navigate to the respective directories to explore the different stages of the RAG pipeline.
