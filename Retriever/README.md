# Retriever

This directory contains implementations for retrieving relevant documents or information based on a query.

## Components

- **[vector_store/](./vector_store/):** Retrieving data from the local vector database.
- **[wikepedia/](./wikepedia/):** Using external APIs (like Wikipedia) as a retrieval source.

## Notebooks

- `vector_store/test.ipynb`: Testing retrieval performance and accuracy from the vector store.
- `wikepedia/code.ipynb`: Implementation of Wikipedia-based retrieval.
