# Chunking Strategies

This directory explores different methods for splitting long documents into smaller, manageable pieces (chunks). Proper chunking is crucial for effective retrieval and context management.

## Strategies

- **[Document/](./Document/):** Generic document chunking.
- **[lengthbased/](./lengthbased/):** Splitting text based on character or token counts.
- **[Semantic_Meaning_Based/](./Semantic_Meaning_Based/):** Using semantic analysis to split text at natural topic transitions.
- **[Text_structure_based/](./Text_structure_based/):** Splitting based on document structure (e.g., headers, paragraphs).
