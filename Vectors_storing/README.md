# Vector Storage

This directory focuses on converting text chunks into vector embeddings and storing them for efficient similarity search.

## Contents

- `code.ipynb`: Main implementation for embedding generation and storage.
- `den.ipynb`: Additional experiments or alternative storage methods.
- **[data/](./data/):** Sample data files used for testing vector storage.
- **[first_db/](./first_db/):** Local storage for the vector database (e.g., ChromaDB).

## Data

The `data/` directory contains sample files like `Al.pdf` and `Newari_Culture.txt` used to populate the vector store.
