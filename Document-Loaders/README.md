# Document Loaders

This directory contains implementations for loading various types of data sources into a format suitable for processing in a RAG pipeline.

## Subdirectories

- **[csv/](./csv/):** Loading data from CSV files.
- **[PyPDF/](./PyPDF/):** PDF document loading using PyPDF and other libraries.
- **[txt/](./txt/):** Simple text file loaders.
- **[web/](./web/):** Web scrapers and loaders for fetching content from URLs.

## Usage

Each subdirectory contains a notebook (`.ipynb`) demonstrating how to use the specific loader.
